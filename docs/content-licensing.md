# Content Licensing Checklist

Track the source and license status for every bundled image, audio file, icon, and illustration before App Store submission.

## Current status

- Emoji-based cards: system emoji, no separate asset files bundled
- Color swatches: inline SVG rectangles created in-app
- First Words image cards: verify source and ownership before shipping to the App Store
- Audio: browser speech synthesis in web; if native audio assets are added later, record ownership here

## Required before release

- Confirm ownership or license for all non-emoji image assets
- Confirm any purchased or generated artwork has redistribution rights
- Keep original source files and receipts in a private asset folder or design repo
